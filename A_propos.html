Notre mission

Qui-sommes nous ?

  • Nous sommes 3 étudiants d'EFREI Paris, école spécialisée dans le numérique. Dans le cadre d'un projet Web, nous avons choisis de confectionner un site pour faire découvrir un pays méconnu de l'Asie : Taïwan. Les membres de l'équipe sont Ulysse DELEPOULLE, Benjamin HOANG et Pierre VISCARDI.

    • Tâches effectuées

  • Pour réaliser ce site web, nous avons d'abord discuté de l'apparence et l'agencement global de notre site. Ensuite, nous nous sommes répartis les tâches méthodiquement. Voici comment le travail a été réparti pour arriver au résultat final :


    • Ulysse DELEPOULLE
    - le header (menu déroulant)
    - le footer (réseaux sociaux)
    - la rubrique "PAYS"
    Benjamin HOANG
    - la page d'accueil (slider)
    - la rubrique "A PROPOS" (formulaire de contact)
